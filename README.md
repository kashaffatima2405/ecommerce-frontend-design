# ecommerce-frontend-design

DevelopersHub Corporation Internship — Frontend Development Task

## Project Overview
A fully responsive **desktop eCommerce website** built with pure HTML5, CSS3, and JavaScript.
Designed based on the Figma file: **Ecommerce Web Design (Community)**.

## Pages
| Page | File | Description |
|------|------|-------------|
| Home Page | `index.html` | Hero slider, categories, featured products, promo banners |
| Product Listing | `pages/products.html` | Filter sidebar, sort bar, product grid, pagination |
| Product Detail | `pages/product-detail.html` | Image gallery, options, reviews, related products |

## Technologies Used
- **HTML5** — Semantic structure (`<header>`, `<nav>`, `<section>`, `<footer>`, `<aside>`)
- **CSS3** — Flexbox, CSS Grid, CSS Variables, animations, hover effects
- **JavaScript** — Tabs, quantity control, size selector, cart badge, scroll animations

## Features
- Sticky header with search bar
- Hero banner with animated dots
- Category grid
- Product cards with hover effects + wishlist
- Filter sidebar with checkboxes, price range slider
- Sort options + grid toggle (3/4/5 columns)
- Product detail with image gallery (thumbnail click to swap)
- Size & color selectors (JavaScript)
- Quantity +/- control
- Tab system (Description / Specs / Reviews / Shipping)
- Star rating reviews with bar chart
- Newsletter form
- Responsive footer with social links

## How to Run
1. Open `index.html` in any browser
2. No server required — pure HTML/CSS/JS
3. All pages are linked together
